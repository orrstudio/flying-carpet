# flying-carpet — Starter (Vite + Three.js)

Простой прототип, показывающий:
- камера движется по spline (CatmullRomCurve3) при скролле страницы;
- инстансированные "облака" (InstancedMesh) для производительного множества объектов;
- базовые свет и материал.

Запуск:
1. npm install
2. npm run dev
3. Открыть http://localhost:5173

Далее:
- заменить простые сферы на glTF модели облаков (DRACO) для качества и размера;
- добавить шейдеры неба/освещения;
- рендер текста через Troika или Three.js TextGeometry;
- оптимизировать LOD/компрессию для мобильных.



    Быстрая настройка: что контролировать

    length — общая длина пути (в примерах length = 140 и т.п.)
    segments — количество контрольных точек (больше — более «зигзагообразно»/подробно)
    lateralAmplitude — насколько сильно путь уходит в стороны
    verticalAmplitude — подъёмы/спуски
    cloudCount — количество облачных групп вдоль пути
    particlesPerCloud — детализация каждой группы (больше = более пушисто, но дороже)
    width — ширина распределения облаков в стороны
    bias — сдвиг распределения влево/вправо (-1..1)
